# Resume Studio 改进路线图

> 本文档描述从当前版本走向成熟商用产品的改进方向与优先级。
> 更新时间：2026-07（P0–P3 已全部落地，详见「现状」；剩余项集中在 P4 商业化）

## 现状（已完成 ✅）

**编辑器（P0）**

- 数据 schema 版本化（v2）+ v1 自动迁移 + 防御性规范化
- 多份简历管理（新建 / 复制 / 重命名 / 删除 / 切换，每份独立模板与排版设置）
- 撤销 / 重做（60 步历史、输入合并、Ctrl+Z / Ctrl+Shift+Z，AI 修改同样可撤销）
- JSON 导入 / 导出（兼容旧版数据格式）
- 自定义板块（证书 / 语言 / 获奖 / 志愿 / 自定义标题，8 套模板原生渲染）
- 轻量富文本（**加粗** / *斜体* / 链接，安全解析非 innerHTML）+ 选区工具条
- 板块与条目拖拽排序（保留箭头按钮作为无障碍回退）

**排版（P1）**

- A4 / US Letter 纸张、页边距三档、字体五档（跟随模板 / 黑 / 宋 / 楷 / 仿宋）
- 一键压缩到一页（有界排版压缩因子）+ 页数指示
- 照片编辑（拖动定位 / 缩放 / 方形裁剪重导出）

**智能（P2，竞争力核心）**

- L1：AI 润色、整份翻译（可撤销）、一句话生成要点、根据简历生成简介 —— 全部带防编造约束与应用前预览
- L2：规则版简历体检（量化率 / 弱动词 / 完整度等，0 AI 成本）+ JD 匹配分析（匹配度 / 缺失关键词 / 建议）
- L3：JD 定制 agent（一键产出岗位定制版新简历副本，原件不动）+ 简历教练 agent（访谈式多轮对话，结构化补丁直接写回简历，可撤销）

**输出与配套（P3）**

- Word（.docx）导出、纯文本导出
- 求职信模块（附加页 + AI 初稿 + 打印分页）
- 预览点击 → 编辑区定位（双向联动）
- 移动端编辑 / 预览标签切换
- PDF 文件名跟随简历名称

**工程与基建（横切 + P4 就绪）**

- 33 个单元测试（vitest）+ 20 个端到端测试（Playwright，覆盖全部核心用户旅程，AI 用路由 mock）
- React 错误边界（模板渲染崩溃不再白屏）
- GitHub Actions CI（单测 + 构建 + e2e）
- Vercel 一键部署就绪：`api/ai/[...path].js` serverless AI 代理已提交（与开发代理同路由同契约）

## 竞争定位（不变）

1. **免费无水印导出**获客（边际成本为零的 local-first 架构）；
2. **隐私即卖点**（简历不上传服务器）；
3. **AI 简历教练**变现（工具卖不上价，服务可以）。AI 是唯一有真实边际成本的能力，也是唯一计费点。

## 剩余工作：P4 商业化（需要产品负责人决策 / 账号资源）

| 事项 | 状态 | 阻塞点 |
| --- | --- | --- |
| 静态部署上线 | 代码就绪（Vercel 配置 + serverless 代理已提交） | 需 Vercel（或其他托管）账号执行部署；国内访问需域名 + ICP 备案主体 |
| AI 代理生产化 | 基础版已提交 | 计量限流 / 免费额度需先有账号体系 |
| 账号 + 支付 + 云同步 | 未开始（按薄后端原则等验证） | BaaS 选型（Supabase / 微信云开发）、支付主体资质 |
| SEO 内容落地页 | 未开始 | 依赖部署与域名 |
| 埋点 / 错误监控 | 未开始 | 供应商选择涉及隐私立场（local-first 卖点），需决策 |

## 明确缓做项（有理由的延期）

| 事项 | 理由 |
| --- | --- |
| 手动分页符 | 连续页预览无法可视化分页效果，需先做真分页预览；条目级防拆已默认生效 |
| 服务端 PDF 渲染 | 需要后端算力（Puppeteer 服务），归入薄后端阶段；当前打印管线在 Chrome/Edge 上质量已达标 |
| ATS 友好度专项检查 | 关键词层面已被 JD 匹配覆盖；格式层面等海外线（Letter 已就绪）有真实用户再做 |
| 模拟面试官 agent | 延伸产品，教练 agent 验证后再扩展 |
| LinkedIn / PDF 解析导入 | 需要服务端解析能力，归入薄后端阶段 |

## 优先级逻辑（回顾）

1. **是不是地基？** → P0 全部完成，后续功能都建立在 schema v2 / 多简历 / 撤销之上（教练补丁可撤销、定制版落到新文档，都是这层地基的直接回报）
2. **要不要后端？** → 所有纯前端项已做完；剩余项全部依赖后端或外部资源，聚焦为一张 P4 决策清单
3. **是不是竞争力？** → 三个差异化支点中，前两个（免费导出、local-first）是架构自带，第三个（AI 教练）已上线 v1
